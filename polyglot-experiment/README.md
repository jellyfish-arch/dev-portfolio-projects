# 🧪 My Polyglot Experiment

A beginner experiment exploring how different programming languages solve simple problems differently!

## 📁 Files

| File | Language | Task |
|------|----------|------|
| `factorial.py` | Python 🐍 | Calculates factorial using recursion |
| `StringReverser.java` | Java ☕ | Reverses a string using a loop |
| `index.html` | HTML + JavaScript 🌐 | Palindrome checker + results page |

## 💡 What I Learned

- Python is clean and great for quick logic.
- Java is structured and strict.
- JavaScript runs directly in the browser.
- Same problem, different syntax — but same logic!.
